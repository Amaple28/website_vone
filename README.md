Este projeto representa a presença digital da **V.ONE**, uma agência de marketing e propaganda de Belo Horizonte especializada em estratégia, criação e performance de alto impacto. O site foi desenvolvido como uma ferramenta de conversão B2B, apresentando a autoridade e os diferenciais da agência através de uma experiência visual imersiva.

## 🌟 Destaques do Desenvolvimento

O site foi construído como uma **Single Page Application (SPA)** com foco em conversão e credibilidade. A estrutura foi pensada para guiar o visitante através de uma jornada estratégica:

1. **Hero Impactante:** Primeira impressão com chamada clara e CTAs duplos.
2. **Prova Social:** Números de sucesso (150+ projetos, 85+ clientes, 10+ anos).
3. **Portfólio de Serviços:** Apresentação visual dos 4 pilares de atuação.
4. **Clientes & Depoimentos:** Validação através de parceiros reais e feedbacks.
5. **Integração WhatsApp:** Formulário que envia dados diretamente para o comercial.

## 🚀 Stack Técnica e Performance

- **Frontend:** HTML5 / CSS3 / JavaScript Vanilla
- **Animações:** AOS (Animate On Scroll) para transições suaves
- **Ícones:** Font Awesome 6.4
- **Tipografia:** Google Fonts (Montserrat + Playfair Display)
- **Responsividade:** Mobile-first com breakpoints otimizados
- **SEO:** Estrutura semântica e meta tags otimizadas

## 🛠️ Funcionalidades Chave

- **Navbar Fixa com Efeito Scroll:** Muda de estilo ao rolar a página
- **Menu Mobile Animado:** Hamburger menu com transições suaves
- **Contador Animado:** Números das estatísticas animam ao entrar na viewport
- **Parallax Sutil:** Background fixo em seções estratégicas
- **Formulário → WhatsApp:** Envia todos os dados preenchidos direto para o WhatsApp
- **Navegação Suave:** Scroll smooth entre seções com offset para navbar fixa
- **Cards Interativos:** Efeitos hover nos serviços e depoimentos

## 📱 Responsividade

O site foi desenvolvido com abordagem mobile-first, garantindo experiência consistente em:
- 📱 Smartphones (320px+)
- 📱 Tablets (768px+)
- 💻 Desktops (1024px+)
- 🖥️ Telas grandes (1200px+)
